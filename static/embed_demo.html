Embeddable Voice AI Agent

Embed the interactive voice agent card on any website with just 2 lines of code!

📦 Quick Embed Code

Copy and paste this snippet into the HTML of any web page right before the </body> tag:

<!-- 1. Voice Agent Custom HTML Tag -->
<agent-stream-voice 
    agent-id="default" 
    server-url="http://localhost:5000">
</agent-stream-voice>

<!-- 2. Async Web Component Script -->
<script src="http://localhost:5000/static/voice-agent-widget.js" async></script>

🚀 How it Works

  • 1

    Native Browser Custom Element

    Uses modern Web Components (Shadow DOM) so its styles and behavior will never interfere with the host website's CSS.

  • 2

    WebAudio Speech Capture

    When the user clicks "Start Conversation", microphone access is requested and 16kHz PCM audio is streamed over WebSockets.

  • 3

    Low-Latency Speech-to-Speech

    Your backend engine processes speech via OpenAI Realtime or Deepgram + Gemini + Cartesia and streams back natural voice responses.
