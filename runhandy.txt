========================================
  AGENT-STREAM-VOICE-AGENT — RUN COMMANDS
========================================

---- SETUP (First Time) ----

# 1. Create virtual environment
python -m venv venv

# 2. Activate virtual environment
#    Windows:
venv\Scripts\activate
#    Mac/Linux:
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Copy env template and fill in secrets
copy env.example .env
# Then open .env and fill in all values


---- RUN LOCALLY ----

# Run the full bot gateway (main entry point)
python main.py

# OR run the API server directly
python main_api.py

# API available at:
#   http://localhost:5002/
#   http://localhost:5002/docs       ← Swagger UI
#   http://localhost:5002/redoc      ← ReDoc
